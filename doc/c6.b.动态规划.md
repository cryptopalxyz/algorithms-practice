#线性动规
买卖股票
121
122
123
188
共同点：任何时刻最多持有1股
不同点：
交易次数限制，1次，2次，k次，无限次
是否有手续费
是否有冷冻期

定义状态：
1。几号？-决定价格
2。当前持仓-决定买还是卖
3。交易了多少次
4。在不在冷冻期
5。手续费是个固定的值，随着交易出现，没必要特别关注，

i：天数
j：持仓 0或1
k：已交易次数
l：是否在冷冻期 0或1
f[i,j]表示i天结束时，持有j股票（0或1）的最大收益
决策<-表示更新max
买 f[i,1] <- f[i-1, 0] - prices[i]
卖 f[i,0] <- f[i-1, 1] + prices[i]
啥也不干 f[i,j] <- f[i-1, j]
边界f[0,0]=0, 其余负无穷
最优性的动态规划问题，数组从1开始算1-n天，0作为边界，没算过/不合法的值为正负无穷，
有助于省去各种复杂的边界判断

目标f[n,0]
时间复杂度O(N)

简化状态转移方程

买卖股票：无交易次数限制，可贪心；交易次数限制，不可贪心。
